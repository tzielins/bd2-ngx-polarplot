Polar plot demo

Simple demo of circadian polar plot. It is not a polar plot in the sense of polar coordinates (radius/angle), but it represents "cyclic data" i.e. modulo domain range.

Generated data always contains fixed two first series: first one demonstrate data with peak times ~(1h+n24h), second a simple data spread.

Petals points to the mean of the input series, their width and radius can be scaled by std dev od the data.
